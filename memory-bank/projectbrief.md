# Project Brief: Hal51 AI Labs Website

## Core Requirements
Create a corporate website for Hal51 AI Labs (lab.hal51.ai) to showcase their mission in AI safety/alignment and education. The site must be hosted on GitHub Pages.

## Goals
- Establish an online presence for Hal51 AI.
- Communicate the company's evolution from LemurBox/Education to AI Safety.
- Highlight the leadership team and their credentials.
- Provide access to research publications.

## Scope
- **Tech Stack**: React, Tailwind CSS, Vite, GitHub Pages.
- **Pages**: Home (Mission/Intro), About (Team), Research (Link).
- **Assets**: Use provided images for founders and product.
